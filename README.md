# Currency Converter API — Прототип решения

## Ссылки
* **Репозиторий проекта на GitHub:** [https://github.com/Petroivch/](https://github.com/Petroivch/)

## Концепция решения
RESTful сервис для получения актуальных курсов валют (включая фиатные деньги и популярные криптовалюты) и выполнения математической конвертации из одной валюты в другую. Сервис принимает запросы в формате JSON и возвращает стандартизированные ответы.

## Спецификация API

| Название действия | Локейшн (URL) | Тип запроса | Описание запроса и ответа |
| :--- | :--- | :--- | :--- |
| **Получить список валют** | `/api/currencies` | `GET` | **Запрос**: не требует параметров.<br>**Ответ**: Возвращает список поддерживаемых валют в формате JSON. |
| **Получить курсы обмена** | `/api/rates` | `GET` | **Запрос**: Query-параметр `base` (опционально, по умолчанию USD). Пример: `/api/rates?base=EUR`.<br>**Ответ**: Возвращает JSON со списком курсов по отношению к базовой валюте. |
| **Сконвертировать валюту** | `/api/convert` | `POST` | **Запрос**: Тело JSON: `{"from": "BTC", "to": "USD", "amount": 0.5}`.<br>**Ответ**: Возвращает JSON с результатом конвертации. |
